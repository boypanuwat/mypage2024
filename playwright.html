คู่มือขั้นตอนการเริ่มใช้ Playwright

คู่มือฉบับย่อนี้จะแนะนำขั้นตอนพื้นฐานในการเริ่มต้นใช้งาน Playwright สำหรับการทดสอบอัตโนมัติ

1. ติดตั้ง Node.js

  • ดาวน์โหลดและติดตั้ง Node.js ได้ที่: Node.js — Download Node.js®
  • ตรวจสอบเวอร์ชันใน Terminal:
node -v
npm -v

2. ติดตั้ง Playwright

  • เปิด Terminal ที่โฟลเดอร์โปรเจกต์ แล้วรันคำสั่ง:
npm install
npx playwright install

หรือถ้า package.json มี Playwright อยู่แล้ว รันแค่:

npm install

3. รัน Test Script

  • 3.1 รันทั้งหมด:
npx playwright test
  • 3.2 รันเฉพาะไฟล์:
npx playwright test tests/example.spec.ts

4. ดูผลลัพธ์บน Terminal

Playwright จะแสดงผลลัพธ์ว่า Test ผ่านหรือไม่ผ่าน พร้อมสรุปจำนวน Test ที่รัน

5. เปิด HTML Report

Playwright จะสร้างไฟล์รายงานในโฟลเดอร์ playwright-report คุณสามารถเปิดดู Report ได้ด้วยคำสั่ง:

npx playwright show-report

ระบบจะเปิด Browser เพื่อแสดง Report ให้คุณดู
